        • void addTransition(S fromState, S toState, E event, groovy.lang.Closure action)

          add an action to this fromState machine

          Parameters:
          fromState - the current state
          toState - the state to go after executing the action. Can be overwritten by the action if it is returning an Enum value
          event - event that triggers the fromState machine
          action - closure that is executed for combination of fromState and event

        • S execute(E event, java.io.Serializable... params)

          execute closure that is identified by currentState and event. After execution, statemachine will be

          • in the following state as defined in addTransition method, if closure returns no object of type S
          • in the state returned by the closure.
          • If no following state is defined, statemachine will stay in currentState.
          Returns:
          the current state after execution
          Parameters:
          event - triggering event
          params - optional parameter to closure. Caution, closure will receive an Object[] Array
